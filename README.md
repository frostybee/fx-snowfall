# Snowfall Simulation

An application that simulates a snowstorm written in JavaFX.

## How to Run the Application

1. Clone this repository
2. Open the project in your preferred IDE
3. **Or** run the following command on Windows
   
```bash
gradlew run
```
On Linux/Mac
   
```bash
./gradlew run 
```
